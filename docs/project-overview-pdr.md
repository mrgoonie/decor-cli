# Project Overview PDR

## Outcome
Create an AI-agent-friendly CLI and MCP server for decorating screenshots, images, and videos.

## Requirements
- Preserve input aspect ratio.
- Support path, URL, and Base64 inputs.
- Support backgrounds, gradients, rounded container, shadows, padding, crop, alignment, text, arrows, shapes, counters, spotlight, quality, and output location.
- Provide templates and default-template configuration.
- Ship CI and semantic-release lanes for `main` and `dev`.

## Non-goals
- Browser UI.
- Animated annotation timelines.
- Bundled static ffmpeg binaries.
