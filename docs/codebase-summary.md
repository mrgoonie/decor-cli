# Codebase Summary

- `packages/core`: render contracts, templates, input safety, media probing, geometry, overlays, image/video rendering.
- `packages/cli`: `decor` command surface.
- `packages/mcp`: `decor-mcp` server with stdio, Streamable HTTP, and SSE compatibility.
- `scripts`: smoke checks and release helpers.
- `claude/skills/decor-cli`: companion agent skill.
